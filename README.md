## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/abhi4418/MysteryMessage.git
   cd MysteryMessage
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the project:
   ```bash
   npm start
   ```
